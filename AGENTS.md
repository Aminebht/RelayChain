## Design Context

### Users
Primary users are marketplace participants who need dependable parcel operations in real transaction flows. They use the platform to coordinate shipment state changes and escrow-backed delivery outcomes without ambiguity.

### Brand Personality
The interface should feel precise, trustworthy, and premium. It should communicate financial-grade clarity, operational control, and confidence at a glance.

### Aesthetic Direction
Light-mode-first interface with restrained, high-craft visual language. No decorative icons and no decorative gradients. Use strong typography hierarchy, deliberate spacing rhythm, and a refined neutral palette with purposeful accents for state and action clarity.

### Design Principles
1. Prioritize decision clarity over decoration in every screen region.
2. Make parcel lifecycle and escrow actions scannable in under 3 seconds.
3. Use visual hierarchy to direct attention to risk, status, and next action.
4. Keep the system visually premium through disciplined type, spacing, and color restraint.
5. Ensure interaction confidence with accessible contrast, clear focus states, and predictable behavior.
